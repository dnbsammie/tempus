<h2 id="about-the-project">🎯 About The Project</h2>
<p>Designed to organize tasks, define times and breaks, intended for software developers or technology lovers who are not looking for something very complex and easy to use.</p>


<h3 align="left">👨🏽‍💻 Tech Stack</h3>

<p align="center">
  <a href="https://skillicons.dev">
    <img src="https://skillicons.dev/icons?i=bash,git,github,md,rust,githubactions&theme=dark&perline=3" alt="icons"/>
  </a>
</p>

<ol>
    <li><b>Event control:</b> Task execution using Pomodoros and Eisenhower Matrix.</li>
    <li><strong>Session control:</strong> Support for serializable state to resume sessions</li>
    <li><strong>Simple persistence: </strong> Initially SQLite (rusqlite) or JSON; migratable later.</li>
</ol>
